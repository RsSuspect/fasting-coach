FASTING COACH — IPHONE SETUP

Option A: Open directly
1. Put these files on any basic web host.
2. Open index.html in Safari.
3. Tap Share.
4. Tap “Add to Home Screen”.

Option B: Use from your computer
1. Open index.html in a browser.
2. The app stores all data locally in the browser.

Important:
- Weight, checklist, and water data are stored only on your device/browser.
- Clearing Safari website data can erase saved entries.
- This is a planning and tracking tool, not medical advice.

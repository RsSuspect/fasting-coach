# Fasting Coach Roadmap

## Guiding Philosophy

Build features in layers.

Each version should feel complete on its own.

Do not begin the next major phase until the previous one is polished and stable.

---

# Version 1.x (Current)

## Completed

- Fasting timer
- Weight tracking
- Water tracking
- Daily checklist
- Settings
- Theme support
- Offline support
- Import / Export
- GitHub deployment
- Cloudflare Pages deployment

---

## Current Goal

Transform the app from a fasting tracker into a personal nutrition coach.

---

# Version 2.0 — Nutrition Foundation

Priority: Highest

## Goals

- Nutrition Profile
- Dynamic calorie target
- Protein target
- Meals per day
- Activity level
- Nutrition summary
- Dashboard calorie card

Success criteria:

The app calculates realistic daily targets and displays them on the Today screen.

---

# Version 2.1 — Food Logging

Goals

- Add Food
- Food diary
- Recent foods
- Favourite foods
- Custom foods
- Meal totals
- Daily nutrition totals

Success criteria:

Logging a common meal takes less than 15 seconds.

---

# Version 2.2 — Recipe System

Goals

- Recipe database
- Saved recipes
- Recipe nutrition
- Add recipe to diary

Success criteria:

Users can build meals from recipes rather than individual foods.

---

# Version 2.3 — Meal Planner

Goals

- Suggested meals
- Remaining calorie calculation
- Remaining protein calculation
- Weekly meal planning
- Shopping list

Success criteria:

The app recommends meals that fit today's remaining targets.

---

# Version 2.4 — Coach

Goals

Rule-based coaching

Examples

- Water reminder
- Protein reminder
- Goal pace warning
- Weight trend encouragement
- Weekly summary

Success criteria:

Every day the app provides one useful recommendation.
